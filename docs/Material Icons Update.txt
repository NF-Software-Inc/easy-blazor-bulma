The following links can be used to download a copy of the Woff2 edition of Material Symbols Outlined.

https://fonts.googleapis.com/css2?family=Material+Symbols+Outlined
https://github.com/google/material-design-icons/raw/master/variablefont/MaterialSymbolsOutlined%5BFILL,GRAD,opsz,wght%5D.woff2
https://raw.githubusercontent.com/google/material-design-icons/master/variablefont/MaterialSymbolsOutlined%5BFILL%2CGRAD%2Copsz%2Cwght%5D.codepoints

Then rename the downloaded file to the desired name.

Then upload that into the following site to convert to otf.
https://cloudconvert.com/woff2-to-woff

Optionally, upload that into font squirrel to build a bundle.
https://www.fontsquirrel.com/tools/webfont-generator

Use the following options
+ Expert
+ WOFF, WOFF2
+ Keep Existing
+ Fix GASP Table
+ No Adjustment
+ Spaces, Hyphens
+ None
+ No Subsetting
+ Clear the -webfont box
